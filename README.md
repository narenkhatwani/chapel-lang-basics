Download Chapel from [link](https://chapel-lang.org/download.html)


# How to compile and run a Chapel Code

### Step 1:

Running Chapel

```bash
source [Enter your path]/chapel-1.24.1/util/setchplenv.bash
``` 

### Step 2:

Compiling the Code

```bash
chpl -o program program.chpl;
```

### Step 3:

Running the Code

```bash
./program
```
